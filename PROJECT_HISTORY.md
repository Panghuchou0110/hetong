当前项目功能（整体概览）
- 手机回收订单管理系统（含合同生成 + 订单管理）。
- 合同生成：粘贴客户信息，自动识别填充，生成 .docx。
- 订单功能：新增/编辑/删除，订单状态，筛选，批量操作，导入/导出 CSV。
- 登录系统：账号登录，重要操作需管理员二次验证。
- 数据库：SQLite 存储，订单为 JSON，配置存 state 表。

服务器操作（简单步骤）
1) 先用 SSH 登录服务器。
2) 进入项目目录：cd /root/hetong
3) 启动项目（示例）：pm2 start app.js --name hetong
4) 查看运行状态：pm2 list
5) 查看日志：pm2 logs hetong

数据安全说明（必须看）
- 数据库文件在：/root/hetong_data/orders.sqlite
- 本次修改没有改数据库结构。
- 上线前一定备份数据库：
  把 /root/hetong_data/orders.sqlite 复制到安全位置。
